Hello Table

A restaurant reservations site built with Rails, JavaScript, React, Redux.

An OpenTable inspired clone utilizing RESTful, AGILE, and TDD principles. The backend for this web app was built with Ruby on Rails. Rails is responsible for handling models and ActiveRecord Associations. The front end is comprised of React components working together on a Flux architecture.

  • Dynamic search using AJAX requests for a streamlined user experience.

  • Capitalized on Rails associations to minimize database queries.

  • Custom front-end React components that provide users with quality defaults and smooth, simplified functionality.
